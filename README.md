# 📖 Giới thiệu

**Electronic Engineering Assistant** là hệ thống hỗ trợ học tập và nghiên cứu trong lĩnh vực Điện tử – Vi xử lý – Embedded Systems.

Project tích hợp nhiều công cụ kỹ thuật phục vụ:

- 🔌 Tính toán linh kiện điện tử
- 📈 Mô phỏng mạch điện
- 📡 Giao tiếp phần cứng
- 🖥 Embedded Monitoring
- ⚙ Điều khiển thiết bị ngoại vi
- 📊 Visualization & GUI

Hệ thống phù hợp cho:

- Sinh viên Điện tử – Viễn thông
- Embedded Developer
- Firmware Engineer
- Hardware Engineer
- FPGA / IoT Learner

- Theo dõi CPU Raspberry Pi
